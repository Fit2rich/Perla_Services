# Perla Homes Services Proposal Generator

A Python-based GUI application for generating personalized service proposals for short-term rental properties managed by **Perla Homes**. The program creates a professional Word document including management and cleaning services, pricing, and company branding.

---

## Features

- **Graphical User Interface (GUI)** powered by `Tkinter`.
- **Service Selection**: Choose from a variety of property management and cleaning services.
- **Customizable Pricing**: Enter percentage fees, visit costs, cleaning costs, and optional photography prices.
- **Automatic Word Document Generation**: Uses `python-docx` to generate a professional proposal with:
  - Company logos
  - Property information
  - Selected services and descriptions
  - Financial breakdown
  - Footer with company contact details
- **Select All** option for quick selection of services or cleaning tasks.
- **VAT inclusion** checkbox for automatic pricing adjustment.
